# Innovate Landing Page

A modern, fancy landing page built with React and TailwindCSS featuring smooth animations and a gradient design aesthetic.

## Features

- **Modern Design**: Gradient colors and floating animations
- **Responsive Layout**: Works perfectly on all device sizes
- **Smooth Animations**: Float effects and fade-in transitions
- **Component-Based**: Clean, modular React components
- **SEO Optimized**: Meta tags for search engines and social media

## Structure

- `index.html` - Main HTML file with dependencies and styles
- `app.js` - Main application component with error boundary
- `components/` - Reusable UI components:
  - `Header.js` - Navigation header with scroll effect
  - `Hero.js` - Hero section with headline and CTA
  - `Features.js` - Features showcase with cards
  - `CTA.js` - Call-to-action section
  - `Footer.js` - Footer with links and copyright

## Color Scheme

- Primary: #6366f1 (Indigo)
- Secondary: #8b5cf6 (Purple)
- Accent: #ec4899 (Pink)
